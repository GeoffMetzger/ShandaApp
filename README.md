# ShandaApp

A personal assistant application engineered with a focus on web accessibility, structured data, and advanced AI-driven programming integrations. 

## 🚀 Overview & Goals
* **Web Migration:** Replatforming core application features entirely to the web.
* **AI Integration:** Implementing an intelligent companion layer to assist directly with programming, context management, and development workflows.
* **Core Tech Stack:** C#, ASP.NET Core MVC, JavaScript, HTML5, CSS3.

## 🛠️ Architecture
The project follows a clean, decoupled MVC (Model-View-Controller) architecture:
* **Controllers/** – Application lifecycle and request handling.
* **Models/** – Core data structures and state representation.
* **Views/** – Web user interface components.
* **Services/** – Business logic and integration layers (including upcoming AI components).
* **wwwroot/** – Static assets and client-side scripts.

## ⚡ Key Features
* **Adaptive Voice & Mood System:** Built-in voice greeting handling paired with dynamic mood response logic and client-side JavaScript fallbacks.
* **Service-Driven Design:** Modular service layers tailored for rapid feature expansion and API integrations.
